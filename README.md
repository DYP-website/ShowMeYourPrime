# Show Me Your Prime Another Time

App statica gratuita per diario alimentare, profilo, calendario palestra, ricettario e progressi.

## Come pubblicarla su GitHub Pages

1. Cancella i vecchi file dalla repository.
2. Carica tutti i file di questa cartella nella root della repository, senza sottocartelle.
3. Vai in **Settings → Pages**.
4. Source: **Deploy from a branch**.
5. Branch: **main**.
6. Folder: **/root**.
7. Salva.

## Salvataggio dati

I dati vengono salvati nel browser con `localStorage`.

- Se apri il sito dallo stesso dispositivo/browser, ritrovi tutto.
- Se un’altra persona apre lo stesso link, non vede i tuoi dati.
- Per spostare i dati usa **Backup dati** e poi **Importa**.
